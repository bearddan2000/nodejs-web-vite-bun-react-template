# nodejs-web-vite-bun-react-template

## Description
A react template for the vite packager.

## Tech stack
- bun
- npm
- vite
- react

## Docker stack
- node

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
